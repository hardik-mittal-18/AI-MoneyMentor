Design a modern, professional AI-powered fintech web application called “AI Money Mentor”.

GOAL:
Create a clean, intelligent dashboard-driven website that helps users analyze their financial health, generate investment plans, and interact with an AI financial advisor.

STYLE:

* Modern fintech UI (similar to Zerodha, Groww, Cred)
* Minimal, clean, premium look
* Light theme with white background + blue/green accents
* Use cards, charts, and soft shadows
* Rounded corners, smooth spacing
* Responsive (desktop + mobile)

---

PAGES / SECTIONS:

1. LANDING PAGE

* Hero section:

  * Headline: “Your AI Financial Advisor for Smarter Investing”
  * Subtext: “Plan, save, and grow your wealth with AI-powered insights”
  * CTA button: “Get Started”
* Features section:

  * Money Health Score
  * AI Investment Planner
  * Smart Budgeting
  * AI Chat Advisor
* How it works (3 steps):

  * Enter details → Analyze → Get plan
* Testimonials (mock)
* Footer with links

---

2. USER INPUT PAGE (ONBOARDING FORM)

* Clean form UI with card layout
* Fields:

  * Age
  * Monthly Income
  * Monthly Expenses
  * Current Savings
  * Financial Goals (optional)
* CTA: “Analyze My Finances”

---

3. DASHBOARD (MAIN SCREEN)

* Top summary cards:

  * Money Health Score (big highlight)
  * Monthly Savings
  * Recommended SIP
* Charts:

  * Pie chart → Income vs Expenses vs Savings
  * Line chart → Future wealth growth
* Sections:

  * Budget Breakdown
  * Investment Allocation (Equity, Debt, Savings)
* Insights panel:

  * “You are overspending in X”
  * “Increase savings by ₹X”

---

4. AI ADVISOR CHAT PAGE

* Chat UI (like ChatGPT)
* Placeholder prompts:

  * “Can I afford a car?”
  * “How much should I invest monthly?”
* AI responses shown in clean message bubbles

---

5. MONEY HEALTH SCORE DETAIL PAGE

* Score out of 100 with visual gauge
* Breakdown:

  * Emergency Fund Score
  * Savings Score
  * Debt Score
  * Investment Score
* Suggestions for improvement

---

6. NAVIGATION

* Sidebar or top navbar:

  * Dashboard
  * Chat Advisor
  * Reports
  * Profile

---

7. EXTRA UI ELEMENTS

* Use icons for finance (₹, charts, wallet)
* Add loading animation when analyzing data
* Add micro-interactions (hover effects)

---

OUTPUT REQUIREMENTS:

* High-fidelity UI design
* Clean spacing and alignment
* Use realistic sample data (₹ values)
* Include charts and dashboard visuals
* Make it look like a real startup product

---

TONE:
Professional, intelligent, trustworthy fintech platform for Indian users.

---

END GOAL:
The design should look like a real product ready for demo in a hackathon and impress judges visually and functionally.
